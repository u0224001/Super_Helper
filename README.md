###專題內容描述###

結合月曆及代辦事項的小幫手，內含記帳功能，能夠簡單統計收支。<br>
是個多功能型的日常小幫手。

----------

- 標題：抽拼圖系統<br>
- 敘述：獲得拼圖。<br>

- 估計天數：5天（規劃1.5天，編碼1.5天，除錯測試2天）<br>

- 估計天數：6天（規劃1.5天，編碼2天，除錯測試2.5天）<br>
- 優先性：5<br>
